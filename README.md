# PolarUI React

Tailwind-first React components by PolarNL.

## Install

```bash
pnpm add @polarnl/polarui-react
```

Required peer dependencies:

- `react`
- `react-dom`
- `tailwindcss`
- `framer-motion`

## Usage

```tsx
import { Button, Input, Dropdown, Checkbox } from '@polarnl/polarui-react';
```

## Components

- `Button`
- `Input`
- `Dropdown`
- `Checkbox`

## Development

```bash
pnpm install
pnpm storybook
pnpm build
```
